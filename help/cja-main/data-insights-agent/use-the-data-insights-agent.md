# [!DNL Data Insights Agent] 사용

자연어 프롬프트를 사용하여 데이터를 분석하고, 자유 형식 테이블, 시각화 및 데이터 비교를 생성하여 데이터 분석가에 대한 의존도를 줄입니다. 이 AI 기반 에이전트는 트렌드와 성과에 대한 빠른 인사이트를 제공하여 이해관계자와 지체 없이 공유할 수 있는 실행 가능한 인사이트를 제공합니다.
