# 전체 테이블 내보내기 관리

Customer Journey Analytics에서 전체 테이블 내보내기를 관리하고 관리하는 방법에 대해 알아봅니다.  이러한 기능을 통해 CJA 관리자는 감독을 간소화하고, 수동 검사를 줄이고, 자동화를 향상시키며, 클라우드 대상에 데이터를 안정적으로 전달할 수 있습니다. 자세한 내용은 [설명서](https://experienceleague.adobe.com/en/docs/analytics-platform/using/cja-workspace/export/export-cloud){target="_blank"}를 참조하세요.
