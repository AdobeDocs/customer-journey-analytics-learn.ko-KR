# 대상 분석 시작

Audience Analysis를 통해 마케터가 Experience Platform에서 대상 세그먼트를 수집하고 채널 전반의 실제 고객 행동을 분석하여 더 나은 캠페인 최적화를 수행하는 방법을 살펴봅니다. 대상 멤버십 데이터를 활성화하고, 사전 설치된 템플릿을 탐색하며, 개인화, 전환 및 ROI를 이끄는 실행 가능한 통찰력을 발견하기 위한 간단한 설정 단계에 대해 알아봅니다.

자세한 내용은 [설명서](https://experienceleague.adobe.com/ko/docs/analytics-platform/using/cja-connections/audience-analysis/audience-analysis-overview){target="_blank"}를 검토하십시오.
