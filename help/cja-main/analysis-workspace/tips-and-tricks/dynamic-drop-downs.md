# Analysis Workspace에서 동적 드롭다운 필터 만들기

전체 차원을 기반으로 드롭다운 필터를 만드는 방법에 대해 알아봅니다. 차원 항목은 보고서 패널에서 사용된 날짜 범위와 동일한 패널의 다른 드롭다운 필터에서 선택한 값을 기준으로 동적으로 업데이트됩니다.
