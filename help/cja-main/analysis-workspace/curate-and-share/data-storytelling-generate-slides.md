# Data Storytelling: 슬라이드 프레젠테이션 생성

Analysis Workspace 프로젝트를 전문 PowerPoint 프레젠테이션으로 변환하고 조정된 시각화, 주요 통찰력 및 매력적인 서사와 함께 생성된 AI를 사용하는 Adobe Customer Journey Analytics의 강력한 [!DNL Data Storytelling] 기능을 살펴보십시오. 이 시간 절약 도구는 고객 여정 트렌드, 전환 드라이버 및 실행 가능한 권장 사항을 이해 당사자와 빠르게 공유 하는 데 도움이 됩니다.

자세한 내용은 Content Analytics [설명서](https://experienceleague.adobe.com/ko/docs/analytics-platform/using/cja-workspace/curate-share/generate-slides){target="_blank"}를 참조하십시오.
