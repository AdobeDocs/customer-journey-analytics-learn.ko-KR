# Analysis Workspace에서 구성 요소 드롭다운 사용

구성 요소 드롭다운을 사용하여 호환 가능한 구성 요소 목록에서 선택하여 시각화를 빌드합니다. 필요한 항목을 찾지 못한 경우 드롭다운에서 바로 새 구성 요소를 만들 수도 있습니다. 고급 사용 사례가 추가되면 이러한 공백은 드롭 영역으로 계속 작동합니다.
