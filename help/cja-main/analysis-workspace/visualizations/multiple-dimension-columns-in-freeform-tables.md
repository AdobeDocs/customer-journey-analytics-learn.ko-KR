# 자유 형식 테이블의 여러 차원 열

CJA의 여러 Dimension 열 기능을 사용하면 자유 형식 테이블을 더욱 강력하게 만들 수 있습니다. 표준 차원 분류 외에도 차원 데이터를 나란히 분석하여 관계형 쌍을 생성하여 데이터를 더 잘 나타내고 이해할 수 있습니다. 또한 고급 정렬 기능을 사용하면 원하는 방식으로 테이블을 조작하여 데이터를 표시할 수 있습니다. 자세한 내용은 [설명서를 참조하십시오.](https://experienceleague.adobe.com/en/docs/analytics-platform/using/cja-workspace/visualizations/freeform-table/freeform-table-multidimensions){target="_blank"}
