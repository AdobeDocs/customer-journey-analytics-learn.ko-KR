# [!DNL Customer Journey Analytics]에 대한 원격 MCP 서버 개요

[!DNL Claude], [!DNL ChatGPT] 또는 [!DNL Cursor]과(와) 같은 AI 도구를 Adobe Customer Journey Analytics(CJA) MCP 서버에 연결하여 데이터 보기를 탐색하고 보고서를 실행하며 자연어를 사용하여 경영진용 KPI 인사이트를 생성하는 방법에 대해 알아봅니다. 이 데모는 원격 MCP 서버가 Adobe Experience Platform에서 확장 가능하고, 통제되는 분석 워크플로 및 자동화된 보고를 가능하게 하는 방법을 보여줍니다.

자세한 내용은 Content Analytics [설명서](https://developer.adobe.com/analytics-mcp/docs/cja/){target="_blank"}를 참조하십시오.
