# 이벤트 심도 사용자 정의

Customer Journey Analytics 파생 필드에서 이벤트 깊이를 구성하여 페이지 깊이, 내부 검색 깊이 등에 대한 분석을 사용자 지정하는 방법을 알아봅니다. 이 기능을 통해 사용자는 사용자 행동을 정밀하게 추적할 수 있으므로 데이터 중심의 통찰력을 향상시킬 수 있습니다.
