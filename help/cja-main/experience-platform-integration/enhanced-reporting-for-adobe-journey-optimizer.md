# 향상된 Adobe Journey Optimizer의 보고 기능

Adobe Journey Optimizer에서 새로운 보고 및 오케스트레이션 측정 기능을 사용할 수 있습니다. 또한 Journey Optimizer의 향상된 기능에는 추가 데이터 소스, 고급 사용자 지정, 데이터 조작, 분석 등을 통해 Customer Journey Analytics와의 상호 운용성 향상 및 가치 상승이 포함됩니다. 자세한 설명서는 [새 보고 인터페이스 시작](https://experienceleague.adobe.com/ko/docs/journey-optimizer/using/channel-report/report-gs-cja){target="_blank"}을 참조하십시오.
