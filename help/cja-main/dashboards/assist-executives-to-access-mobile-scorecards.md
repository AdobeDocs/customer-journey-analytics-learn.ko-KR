# 경영진이 모바일 스코어카드에 액세스할 수 있도록 지원

경영진이 [!DNL Analytics] 대시보드 모바일 애플리케이션을 사용하여 [!DNL Customer Journey Analytics]에서 생성된 모바일 스코어카드에 액세스할 수 있도록 지원합니다. 모바일 스코어카드는 언제든지 핵심 성과 지표 및 세부 보기 또는 분류에 대한 액세스를 제공합니다.
