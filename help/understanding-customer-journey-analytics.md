# Customer Journey Analytics 이해

Adobe Customer Journey Analytics(CJA)는 Adobe Experience Platform을 기반으로 구축된 응용 프로그램 서비스입니다. 플랫폼 데이터 집합에 대해 멀티채널 분석을 수행할 수 있도록 Analysis Workspace라는 리치 분석 도구를 플랫폼에 제공합니다.
